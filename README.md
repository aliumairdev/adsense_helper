# adsense_helper

Helper for Google Adsense code

## Requirements

* Ruby ~> 2.0

## Installation

Include the gem in your Gemfile:

```ruby
gem 'adsense_helper'
```

## Usage

```ruby
adsense_tag(client: 0000000000000000, slot: 1111111111, dimension: '336x280')
```
